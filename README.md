# 🏠 HomiAI

AI-Powered Interior Design Assistant built using **Google Gemini Vision**, **Flask**, and **Python**. Upload a room image and receive professional AI-generated interior design recommendations with downloadable PDF reports.

---

## ✨ Features

- 📷 Upload room images
- 🤖 AI-powered room analysis using Gemini Vision
- 🎨 Color palette recommendations
- 🛋 Furniture suggestions
- 💡 Lighting improvements
- 🌿 Decor recommendations
- 💰 Estimated renovation budget
- 📄 Download professional PDF reports
- 📚 Design history
- 🔐 User Login & Registration

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/KarthikChowdary24/HomiAI.git
```

Go to project

```bash
cd HomiAI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GEMINI_API_KEY=your_api_key
GROQ_API_KEY=your_api_key
```

Run the project

```bash
python app.py
```
